INNOVATION

What Makes BioCascade Different?

Moving beyond single-biomarker approaches to multi-system cascade modeling

Cascade Interactions

Engineered features capturing multi-system dysfunction: Vascular-Metabolic Load (BP × HbA1c) and Metabolic-Renal Stress (HbA1c × 1/eGFR) rank as top predictors, validating that interactions outperform individual biomarkers.

Novel Contribution

Full Interpretability

Every prediction explained via SHAP values. Clinicians see exactly which biomarkers drive risk for each patient, enabling targeted interventions (e.g., prioritize glucose control vs BP management).

Clinical Trust

Two-Model Validation

Model A (0.998 AUC): Validates phenotype definition. Model B (0.821 AUC): Demonstrates practical screening with independent markers only. This dual approach ensures scientific rigor.

Methodology

Accessible Screening

Model B uses routine metabolic panel + waist measurement (~$50). No expensive imaging required. Enables screening in primary care settings and underserved populations.

Healthcare Equity

Early Detection

Independent markers capture upstream metabolic dysfunction before BP/glucose become severely abnormal. Provides a prevention opportunity window for lifestyle and pharmacologic interventions.

Preventive Medicine

Reproducible Research

Complete pipeline from raw NHANES data to deployment. Includes DuckDB for data management, scikit-learn for modeling, SHAP for interpretability, and Flask web interface.

ARCHITECTURE

The Interpretability Engine

End-to-end pipeline from raw data to clinical insights

Data Pipeline

NHANES 2017-2018 (5,533 patients) → DuckDB storage → Feature engineering (50+ biomarkers) → Label definition (multi-system dysfunction proxy)

ML Training

Random Forest with class balancing → Two-model validation → Feature importance ranking → Cross-validation → Model persistence (joblib)

SHAP Explanations

TreeExplainer for global importance → Waterfall plots for individual patients → Feature contribution analysis → Clinical narrative generation

Technology Stack
